# Tange
Simple todo app in the command line
Currently only supports Linux

## Usage
```
   $ tange add <content>
```
Creates a new todo with the provided content
```
   $ tange list <option>
   $ tange list all
   $ tange list 10
```
List todos based on the option: ```all``` or ```<number>```

```
   $ tange remove <index>
```
Removes a todo with the ```<index>```
